☁️ GitHub Gist Sync
Speichert deine Daten in einem privaten GitHub Gist — erreichbar von jedem Gerät. Token wird nie exportiert.
Setup: 1. Neues privates Gist auf gist.github.com erstellen (Dateiname egal). 2. Token auf github.com/settings/tokens → „Gist"-Berechtigung. 3. Gist-ID aus der URL kopieren (der lange Hash).

📊 CSV-Exporte
Einzelne Bereiche als CSV — für externe Auswertung, Backups oder manuellen Vergleich bei Versionskonflikten.
